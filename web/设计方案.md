# 进销存管理系统 - UI 设计方案

## 📋 方案概述

本文档提供4套不同风格的 UI 设计方案，每套方案包含完整的设计规范和实现指南。

---

## 🎨 方案一：简约商务风（推荐）

### 1. 页面布局结构
- **导航区**：左侧固定侧边栏（64px-200px 可折叠），深色主题
- **内容区**：卡片式布局，12px 间距，12px 内边距
- **功能区**：顶部工具栏，紧凑排列
- **空间分配**：导航占比 15-25%，内容区占比 75-85%

### 2. 色彩系统
```css
/* 主色 */
--primary-50: #eff6ff
--primary-100: #dbeafe
--primary-200: #bfdbfe
--primary-500: #3b82f6
--primary-600: #2563eb
--primary-700: #1d4ed8

/* 辅助色 */
--success-500: #10b981
--warning-500: #f59e0b
--danger-500: #ef4444
--info-500: #06b6d4

/* 中性色 */
--gray-50: #f9fafb
--gray-100: #f3f4f6
--gray-200: #e5e7eb
--gray-300: #d1d5db
--gray-500: #6b7280
--gray-700: #374151
--gray-800: #1f2937
--gray-900: #111827
```

### 3. 排版层级
```css
/* 字体 */
--font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

/* 字号体系 */
--text-xs: 12px
--text-sm: 13px
--text-base: 14px
--text-lg: 16px
--text-xl: 18px
--text-2xl: 20px

/* 行高 */
--leading-tight: 1.25
--leading-normal: 1.5
--leading-relaxed: 1.625

/* 字重 */
--font-normal: 400
--font-medium: 500
--font-semibold: 600
--font-bold: 700
```

### 4. 组件样式
- **按钮**：圆角 6px，高度 32px，字号 13px
- **表单**：圆角 6px，边框 1px，高度 32px
- **卡片**：圆角 8px，阴影 subtle
- **表格**：边框间隔 0，斑马纹浅灰色
- **标签**：圆角 4px，高度 22px，字号 12px

### 5. 交互反馈
- **悬停**：背景色加深 5-10%，平滑过渡 0.2s
- **加载**：骨架屏 + 旋转加载图标
- **过渡动画**：0.2s ease-in-out
- **点击反馈**：缩放 0.98

## 🌟 方案二：清新科技风

### 1. 页面布局结构
- **导航区**：顶部横向导航，高度 56px
- **内容区**：无卡片边界，网格布局，8px 间距
- **功能区**：浮动操作按钮（FAB）
- **空间分配**：导航占比 8-10%，内容区占比 90-92%

### 2. 色彩系统
```css
/* 主色 - 青色渐变 */
--primary-500: #0ea5e9
--primary-600: #0284c7
--primary-gradient: linear-gradient(135deg, #0ea5e9, #06b6d4)

/* 辅助色 */
--accent-500: #8b5cf6
--success-500: #22c55e
--warning-500: #f97316
--danger-500: #f43f5e

/* 中性色 - 浅色调 */
--bg-primary: #ffffff
--bg-secondary: #f8fafc
--border-color: #e2e8f0
--text-primary: #0f172a
--text-secondary: #64748b
```

### 3. 排版层级
```css
/* 字体 */
--font-family: 'Inter', system-ui, sans-serif

/* 字号体系 */
--text-xs: 11px
--text-sm: 12px
--text-base: 14px
--text-lg: 15px
--text-xl: 17px
--text-2xl: 24px

/* 行高 */
--leading-tight: 1.3
--leading-normal: 1.55
--leading-relaxed: 1.7

/* 字重 */
--font-light: 300
--font-normal: 400
--font-medium: 500
--font-semibold: 600
```

### 4. 组件样式
- **按钮**：圆角 8px，高度 36px，字号 14px，渐变背景
- **表单**：圆角 8px，高度 36px，聚焦时发光边框
- **卡片**：圆角 12px，轻微阴影，悬浮时上浮
- **表格**：无边框，分隔线浅色，行高舒适
- **标签**：圆角 20px，胶囊形状

### 5. 交互反馈
- **悬停**：卡片上浮 2px，阴影增强
- **加载**：脉冲动画 + 进度条
- **过渡动画**：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- **点击反馈**：波纹效果

## 🎭 方案三：暗黑科技风

### 1. 页面布局结构
- **导航区**：左侧半透明玻璃态侧边栏，毛玻璃效果
- **内容区**：分层设计，Z 轴空间感，16px 间距
- **功能区**：侧边悬浮工具栏

## 📖 页面类型覆盖

### 1. 列表页（供应商/客户/商品）
- **布局**：顶部工具栏 + 数据表格 + 底部分页
- **特点**：紧凑表格，快速操作
- **适配**：4套方案均适用

### 2. 表单页（新增/编辑对话框）
- **布局**：两列网格，标签 + 输入框
- **特点**：清晰的必填标识，分组布局
- **适配**：4套方案均适用

### 3. 数据可视化页（仪表盘/报表）
- **布局**：统计卡片 + 图表网格
- **特点**：数据密度高，色彩区分
- **适配**：方案一、二、三最佳

### 4. 登录页
- **布局**：居中卡片，品牌展示
- **特点**：简洁聚焦，品牌突出
- **适配**：4套方案均适用
